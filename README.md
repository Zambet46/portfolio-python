# Projetos em Python – Exercícios de Lógica e Programação

Este repositório contém **projetos simples desenvolvidos em Python** com foco em prática de lógica de programação, estruturas de controle, tratamento de erros e resolução de problemas.

Os exemplos simulam cenários práticos, como cálculo de pontuação, manipulação matemática e execução de sorteios, reforçando boas práticas iniciais de programação.

---

## Tecnologias utilizadas
- Python 3.x
- Biblioteca padrão do Python (`math`, `random`)

---

## Projetos disponíveis

### 1. Cálculo de Pontuação – Skatistas (`Funcao.py`)
Projeto que simula o cálculo da nota final de um(a) skatista com base em notas atribuídas por jurados.

**Descrição:**
- Recebe 5 notas digitadas pelo usuário (valores entre 0 e 10)
- Elimina a maior e a menor nota
- Calcula a média das 3 notas restantes
- Exibe o resultado final formatado

**Conceitos trabalhados:**
- Listas
- Funções
- Laços de repetição (`while`)
- Tratamento de exceções (`try/except`)
- Validação de entrada de dados


### 2. Raiz Quadrada de Números (raiz.py)

Projeto que calcula a raiz quadrada de uma lista de números e identifica quais resultados são números inteiros.

Descrição:

Calcula a raiz quadrada de cada número da lista

Exibe os valores com duas casas decimais

Identifica e destaca quais raízes são números inteiros

Conceitos trabalhados:

Laços de repetição (for)

Listas

Operações matemáticas

Uso da biblioteca math


## 3. Sistema de Sorteio Simples (Tarefa.py)

Projeto que simula um sistema de sorteio com entrada do usuário.

Descrição:

Solicita o número de participantes

Valida se a entrada é um número inteiro positivo

Realiza um sorteio aleatório

Exibe o número sorteado

Conceitos trabalhados:

Entrada de dados via terminal (input)

Geração de números aleatórios

Tratamento de exceções

Validação lógica

Execução:

python Tarefa.py
